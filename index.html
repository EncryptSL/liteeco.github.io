LiteEco

Multiple currency minecraft plugin for Paper and Spigot software

Easy to Setup

LiteEco was designed to be easy to use and lightweight, without bloatware, for all regular Minecraft server owners.

Compatibility

Plugin supports Paper and connection to MariaDB, MySQL, SQLite and PostgreSQL.

Powered by Performance

Asynchronous and stores data in memory, which it then writes to the database.
